---
title: "文档描述了可以使用的服务操作"
---

## 理由

集成可以注册服务操作，以提供标准实体无法实现的功能。  
这些服务操作可能比标准服务操作更难使用，因此我们希望确保文档描述它们的功能以及参数是什么。

## 示例实现

```markdown showLineNumbers
## 操作

集成提供以下操作。

### 操作: 获取时间表

`my_integration.get_schedule` 服务用于从集成中获取时间表。

- **数据属性**: `config_entry_id`
    - **描述**: 获取时间表的配置条目的ID。
    - **可选**: 否
```

## 例外

对此规则没有例外。
