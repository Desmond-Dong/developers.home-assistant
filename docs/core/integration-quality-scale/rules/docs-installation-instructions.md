---
title: "文档提供了集成的逐步安装说明，包括（如有需要）先决条件"
---

## 理由

我们希望用户在设置集成时能够顺利进行。
这意味着文档应该提供清晰简洁的说明，指导如何安装该集成。
这包括安装集成所需的任何先决条件。

## 示例实现

```markdown showLineNumbers
## 先决条件

1. 打开应用商店并安装 **MyProduct** 应用。
2. 创建一个账户。
3. 添加设备到应用中。
4. 打开应用并进入 **设置** 页面。
5. 选择 **暴露 API**。

{% include integrations/config_flow.md %}
```

## 例外

这个规则没有例外。
